Callisto Contact Manager

Tech: MySQL • PHP or Node API • HTML/CSS/JS

Callisto is a simple, focused web app where users create an account and securely manage a personal list of contacts. Core features include adding, searching, editing, and deleting contacts with server-side validation.

Project Description

This project implements a basic authentication flow (register/login), hashed passwords, and CRUD endpoints for contacts. The database uses normalized tables and indexes for fast lookups. Frontend forms include accessible labels, helpful validation messages, and keyboard-friendly controls.

Key Features

  • Account creation, login, and logout with session handling
  • Create, read, update, and delete contacts (name, phone, email, notes)
  • Search and filter contacts by name or tag
  • Server-side input validation and error handling

Skills Learned

  • SQL schema design
  • Parameterized queries
  • RESTful API design
  • Form validation
  • Auth & hashing
  • Accessibility basics

My Role & Contributions

I built the database schema (users, contacts), wrote the CRUD endpoints, and implemented the login flow with hashed passwords. I also added search with indexes and improved the UI for error states and form feedback.
