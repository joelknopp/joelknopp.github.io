vibeCheck

Tech: React • Node • Express • MongoDB • Email Verification

vibeCheck is a MERN app where users sign up with email verification, then save, label, and favorite campus locations on an interactive map. It’s designed to help students quickly find study spots and navigate a large campus.

Project Description

The system features a Node/Express API with JWT-based auth, a MongoDB data model for users and places, and a React frontend. Users can create named pins, add notes/tags, and mark favorites. The design supports responsive layouts and keyboard navigation.

Key Features

  • User registration with email verification and secure login
  • Create and manage saved locations with titles, tags, and notes
  • Favorite lists and quick filtering
  • Responsive, accessible UI suitable for mobile and desktop

Skills Learned

  • React state & routing
  • Express APIs
  • MongoDB schema design
  • JWT auth
  • Email verification flow
  • Responsive UI

My Role & Contributions

I implemented the authentication and email verification flow using SendGrid, designed the MongoDB models, implemented CRUD API endpoints, and built the location saving/favoriting features.
